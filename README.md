# Handlebars

### В проекте присутствует следующий функционал:

- Добавление дела
- Изменение статуса дела
- Сортировка по статусу дела
- Сброс сортировки

### В проекте используются следующие технологии:

- HTML
- CSS
- JS
- Node.js
- REST API
- БЭМ
- Express.js
- Mongo.DB
- Handlebars
